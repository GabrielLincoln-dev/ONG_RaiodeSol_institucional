# Site institucional ONG Raio de Sol

Recriação do meu primeiro projeto feito na faculdade, em parceria com a ONG Raio de Sol. O intuito do projeto foi criar um site para melhorar o posicionamento digital da ONG, desenvolvido para a matéria Web Basics, na qual foram ensinados desenvolvimento web com HTML, CSS e JavaScript, bem como React. Infelizmente, o projeto saiu do ar devido à ONG não conseguir pagar o domínio e a hospedagem. Recriei o projeto utilizando Vibe Coding com o Lovable.
